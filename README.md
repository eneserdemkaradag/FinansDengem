# FinansDengem
Future Talent 201 Projesi: FinansDengem. Günlük bütçe (Normal Mod) ile borsa/fiziksel varlıkları (Yatırımcı Modu) birleştiren çift modlu finans uygulamasıdır. İçerisindeki LLM (AI Wrapper) mimarisi, harcamaları otomatik kategorize eder ve bütçe/portföy dengesi için yapay zeka destekli akıllı tavsiyeler sunarak vizyona kat üstüne kat çıkar.
